## Skill 协作优化

### /dev-pipeline 与 /user-manual 的协作

**当前问题**：两个Skill独立运行，用户手册可能过时

**优化方案**（已实施）：
- 人工测试环节强制调用 /user-manual 刷新
- 确保测试引导文档与当前代码一致

### /dev-pipeline 与 /retrospect 的协作

**当前问题**：retrospect依赖dev-pipeline产出，但产出可能不完整

**优化方案**：
- retrospect应检查index.md是否与pipeline-state.yaml一致
- 不一致时自动修正（本次复盘已发现并修正）
