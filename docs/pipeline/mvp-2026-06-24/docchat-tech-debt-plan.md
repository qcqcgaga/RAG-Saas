## 治理计划

### 紧急修复（V1版本必须处理）

#### TD-01: 接入真实Embedding API
- **影响**：RAG对话核心功能完全不可用，向量搜索返回随机结果
- **方案**：接入讯飞Embedding API或第三方（OpenAI/智谱）
- **预估工时**：2-3天
- **验收标准**：文档上传后向量搜索返回语义相关的结果

#### TD-02: 接入真实LLM API
- **影响**：对话功能返回占位文本，核心价值无法体现
- **方案**：接入讯飞Coding Plan API（SSE流式）
- **预估工时**：2-3天
- **验收标准**：用户提问后返回基于文档内容的真实回答

#### TD-05: 集成测试增加真实环境验证
- **影响**：H2+MockBean全绿但PG+Milvus可能崩溃
- **方案**：
  1. 保留H2快速测试（CI环境）
  2. 增加Docker Compose环境集成测试（本地/预发布）
  3. TestContainers或docker compose up后执行测试
- **预估工时**：3-5天
- **验收标准**：集成测试可在真实PG+Redis+Milvus环境中通过

### 一般改进（V1版本尽量处理）

#### TD-03: Widget Token完整验证
- **影响**：当前简化从Header解析tenantId，存在安全风险
- **方案**：实现完整的widget_token生成和验证逻辑
- **预估工时**：1天
- **验收标准**：widget_token通过JWT签名验证，无法伪造

### 持续优化（后续版本按需处理）

| TD-07 | Milvus collection缓存 | 文档上传频率 > 100次/天 时优化 |
