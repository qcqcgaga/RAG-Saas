# 产品概述

## 产品简介

DocChat 是面向独立开发者和小团队的文档智能客服 SaaS 解决方案。只需上传产品文档或 FAQ，即可在 5 分钟内生成可嵌入网站的 AI 聊天组件，让您的用户获得基于文档的即时智能问答体验。

### 核心价值

- **零代码嵌入**：上传文档 → 自动处理 → 复制嵌入脚本 → 聊天组件上线，全程无需编码
- **RAG 可控可测**：文档自动切分向量化，对话回答附带来源引用，评测集让检索质量可量化、可迭代
- **对话即焚安全**：终端访客对话不持久化存储，保护访客隐私，降低数据合规风险
- **用量透明可控**：API Key 鉴权 + 用量统计 + 每日限额，调用情况一目了然

### 适用场景

- **产品官网**：嵌入智能客服，自动回答产品使用问题，减少人工客服成本
- **技术文档站**：为 API 文档、SDK 文档提供智能搜索问答，提升开发者体验
- **FAQ 页面**：将传统 FAQ 升级为交互式问答，用户无需翻阅长列表
- **内部知识库**：团队内部共享文档知识，新成员快速获取信息

## 角色说明

DocChat 定义了三种用户角色：

| 角色 | 说明 | 典型操作 |
|------|------|----------|
| 租户管理员 | 注册账号、创建工作空间 | 管理团队、配置组件、管理 API Key、查看用量、配置 LLM |
| 团队成员 | 被邀请加入工作空间 | 管理知识库文档、配置评测集、调整聊天组件外观 |
| 终端访客 | 在嵌入的聊天组件中提问 | 提问并获得基于文档的回答 |

### 关键技术选型

| 组件 | 技术 | 用途 |
|------|------|------|
| 后端 | Java 21 + Spring Boot 3.3 | API 服务、业务逻辑 |
| 关系数据库 | PostgreSQL 16 | 租户、文档、任务、统计、评测数据存储 |
| 向量数据库 | Milvus 2.4 | 文档向量存储与相似度检索 |
| 缓存/队列 | Redis 7 | 缓存加速、异步任务队列、限额计数 |
| 管理后台 | Vue 3 + Ant Design Vue 4 | 租户管理界面 |
| 聊天组件 | TypeScript (IIFE) | 嵌入式聊天窗口 |
| 大语言模型 | 可配置（默认讯飞 Coding Plan API） | RAG 问答生成 |

## V1 新增功能亮点

### API Key 管理

为聊天组件提供专用的 API Key 鉴权方式，替代旧版 widget_token。每个租户最多可创建 5 个 API Key，支持命名、吊销和每日调用限额。

### 用量统计

全面掌握 API 调用情况。提供 7 天/30 天概览、每日明细和趋势图，仅统计通过 API Key 鉴权的正式对话，预览对话不计入。

### 评测集管理

让 RAG 检索质量可量化。创建评测集、添加问答对（最多 50 对）、批量导入，一键执行评测获取 Hit Rate，历史结果可对比。

### LLM 配置

租户可自定义 LLM API 地址和密钥，支持连通性测试。未配置时自动使用系统默认 LLM，灵活适配不同模型供应商。

### 聊天组件预览

管理后台内嵌预览窗口，实时预览外观配置效果，模拟终端访客问答。预览对话不计入用量统计，对话不持久化。
