# 功能介绍

DocChat V1 版本包含九大功能模块，覆盖从文档上传到 RAG 质量评测的完整链路。

## 功能一览

| 模块 | 核心能力 | 版本 | 快速入口 |
|------|----------|------|----------|
| [租户管理](tenant.md) | 注册登录、团队管理、角色权限 | MVP | 管理后台 → 租户设置 |
| [知识库管理](knowledge.md) | 文档上传、自动切分向量化、版本管理 | MVP | 管理后台 → 知识库 |
| [RAG 对话](chat.md) | 智能问答、来源引用、双鉴权模式 | MVP | 嵌入的聊天组件 |
| [聊天组件](widget.md) | 嵌入脚本、外观配置、品牌定制、预览 | MVP | 管理后台 → 聊天组件 |
| [API Key 管理](apikey.md) | API Key 创建/吊销/重命名、每日限额 | V1 | 管理后台 → API Key |
| [用量统计](stats.md) | 概览、每日明细、趋势图 | V1 | 管理后台 → 用量统计 |
| [评测集管理](eval.md) | 问答对管理、批量导入、Hit Rate 评测 | V1 | 管理后台 → 评测集 |
| [LLM 配置](llm-config.md) | 租户级 LLM 配置、连通性测试 | V1 | 管理后台 → LLM 配置 |

## 功能流转

```
注册/登录 → 上传文档 → 自动切分向量化 → 配置LLM → 创建API Key → 配置聊天组件 → 嵌入网站 → 访客问答
   ↓            ↓             ↓            ↓          ↓            ↓            ↓          ↓
 租户管理    知识库管理     异步任务     LLM配置    API Key管理   聊天组件      嵌入脚本    RAG对话
                                                         ↓
                                                    用量统计(计数)
                                                         ↓
                                                    评测集(质量验证)
```

## 异步任务

文档上传后，系统会自动创建异步处理任务，完成文本提取、切分和向量化。您可以在 **任务** 页面查看处理进度和状态。

> 详细的任务管理说明集成在知识库管理中，任务页面主要用于监控和排查问题。

## V1 版本变更摘要

V1 在 MVP 基础上新增了以下能力：

| 变更 | 说明 |
|------|------|
| 双鉴权模式 | 对话接口支持 API Key（`dc_` 前缀）和 JWT（`eyJ` 前缀）两种鉴权 |
| API Key 管理 | 新增 API Key 模块，支持创建/吊销/重命名，最多 5 个 |
| 用量统计 | 新增统计模块，仅统计 API Key 鉴权的正式对话 |
| 评测集 | 新增评测模块，支持 Hit Rate 评测和结果对比 |
| LLM 配置 | 新增租户级 LLM 配置，未配置时使用系统默认 |
| 组件预览 | 聊天组件页面新增预览窗口，支持模拟对话 |
| 嵌入脚本升级 | 嵌入脚本新增 `data-api-key` 参数，旧版 `data-token` 过渡期兼容 |
