# 功能介绍

DocChat MVP 版本包含五大核心功能模块，覆盖从文档上传到聊天组件嵌入的完整链路。

## 功能一览

| 模块 | 核心能力 | 快速入口 |
|------|----------|----------|
| [租户管理](tenant.md) | 注册登录、团队管理、角色权限 | 管理后台 → 租户设置 |
| [知识库管理](knowledge.md) | 文档上传、自动切分向量化、版本管理 | 管理后台 → 知识库 |
| [RAG 对话](chat.md) | 智能问答、来源引用、流式输出 | 嵌入的聊天组件 |
| [聊天组件](widget.md) | 嵌入脚本、外观配置、品牌定制 | 管理后台 → 聊天组件 |

## 功能流转

```
注册/登录 → 上传文档 → 自动切分向量化 → 配置聊天组件 → 嵌入网站 → 访客问答
   ↓            ↓             ↓               ↓            ↓          ↓
 租户管理    知识库管理     异步任务         聊天组件      嵌入脚本    RAG对话
```

## 异步任务

文档上传后，系统会自动创建异步处理任务，完成文本提取、切分和向量化。您可以在 **任务** 页面查看处理进度和状态。

> 详细的任务管理说明集成在知识库管理中，任务页面主要用于监控和排查问题。
