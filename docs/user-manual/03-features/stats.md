# 用量统计

> V1 新增功能

## 功能概述

用量统计模块帮助您全面掌握 API 调用情况，包括对话调用量、Token 消耗和对话数。提供概览、每日明细和趋势图三种视角，支持 7 天和 30 天两种统计周期。

> **重要**：用量统计仅统计通过 **API Key 鉴权**的正式访客对话。管理后台预览对话（JWT 鉴权）不计入统计。

## 统计指标

| 指标 | 说明 |
|------|------|
| 调用次数（calls） | 通过 API Key 鉴权的对话请求总次数 |
| Token 消耗（tokens） | 对话过程中 LLM 消耗的 Token 总数 |
| 对话数（conversations） | 独立对话会话数量 |

## 用量概览

### 操作步骤

1. 登录管理后台，进入 **用量统计** 页面
2. 默认显示 **7 天概览**，可切换为 **30 天概览**

[截图：用量统计 - 概览页面]

### 概览数据

概览页面展示选定周期内的汇总数据：

| 数据项 | 说明 |
|--------|------|
| 总调用次数 | 周期内 API Key 鉴权的对话请求总数 |
| 总 Token 消耗 | 周期内 LLM 消耗的 Token 总数 |
| 总对话数 | 周期内独立对话会话数 |
| 日均调用 | 平均每日调用次数 |
| 日均 Token | 平均每日 Token 消耗 |

### 切换统计周期

在概览页面顶部，点击 **7 天** 或 **30 天** 切换统计周期：

- **7d**：统计最近 7 天的数据
- **30d**：统计最近 30 天的数据

[截图：周期切换按钮]

## 每日明细

### 操作步骤

1. 在用量统计页面，切换到 **每日明细** 标签
2. 选择日期范围（起始日期和结束日期）
3. 查看每日统计数据

[截图：每日明细页面]

### 明细数据

每日明细以表格形式展示，每行包含：

| 列 | 说明 |
|----|------|
| 日期 | 统计日期 |
| 调用次数 | 当日 API Key 鉴权的对话请求次数 |
| Token 消耗 | 当日 LLM 消耗的 Token 数 |
| 对话数 | 当日独立对话会话数 |

### 日期范围选择

- 点击日期选择器，选择起始日期和结束日期
- 日期范围最长 30 天
- 默认显示最近 7 天

## 趋势图

### 操作步骤

1. 在用量统计页面，切换到 **趋势图** 标签
2. 选择统计周期和指标
3. 查看趋势变化

[截图：趋势图页面]

### 趋势图配置

| 配置项 | 选项 | 说明 |
|--------|------|------|
| 统计周期 | 7d / 30d | 趋势图的时间范围 |
| 指标 | calls / tokens / conversations | 趋势图展示的指标 |

### 趋势图解读

- **X 轴**：日期
- **Y 轴**：选定指标的数值
- **数据点**：每日的指标值
- **趋势线**：连接各数据点，展示变化趋势

通过趋势图可以：
- 发现调用量的周期性波动
- 识别异常峰值或低谷
- 评估知识库更新对调用量和 Token 消耗的影响

## 统计原理

### 数据采集

用量统计通过 AOP 切面自动采集，流程如下：

```
API Key 鉴权对话请求
        ↓
ChatStatAspect 拦截
        ↓
记录 tenantId + apiKeyId + authType
        ↓
对话完成后统计 calls + tokens + conversations
        ↓
写入统计表（按日聚合）
```

### 统计范围

| 鉴权方式 | 是否统计 | 说明 |
|----------|----------|------|
| API Key（`dc_` 前缀） | 统计 | 正式访客对话 |
| JWT（`eyJ` 前缀） | 不统计 | 管理后台预览对话 |

### 数据粒度

- 统计数据按 **日** 聚合
- 每日数据在次日零点后可查
- 统计数据按租户隔离

## 与 API Key 的关系

用量统计与 API Key 管理紧密关联：

1. **统计前提**：只有通过 API Key 鉴权的对话才计入统计
2. **限额监控**：通过用量统计了解调用量，判断是否接近每日限额
3. **Key 追踪**：统计数据按 apiKeyId 记录，可追溯具体 Key 的使用情况

详见 [API Key 管理](apikey.md)。

## 注意事项

- 统计数据存在短暂延迟（通常不超过 5 分钟），非实时数据
- 预览对话（管理后台模拟测试）不计入统计，不影响限额
- 统计数据按自然日聚合，不支持小时级粒度
- 删除 API Key 后，该 Key 的历史统计数据仍可查看
