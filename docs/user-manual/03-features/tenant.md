# 租户管理

## 功能概述

租户管理模块负责用户注册登录、工作空间管理和团队成员协作。每个注册用户自动创建一个租户工作空间，数据严格隔离。

## 注册与登录

### 注册新账号

1. 打开管理后台登录页面
2. 点击 **注册** 链接
3. 填写以下信息：
   - **邮箱**：用于登录和接收团队邀请，必须唯一
   - **密码**：至少 8 位，必须包含大写字母、小写字母和数字
   - **租户名称**：您的工作空间显示名称，如"我的产品团队"
   - **租户标识（slug）**：工作空间唯一标识，仅支持字母、数字和连字符，如 `my-product`
4. 点击 **确认注册**

[截图：注册页面]

注册成功后，系统自动登录并跳转到知识库页面。

### 登录

1. 打开管理后台登录页面
2. 输入 **邮箱** 和 **密码**
3. 点击 **登录**

[截图：登录页面]

> **注意**：登录后系统签发 JWT Token，有效期 24 小时（86400 秒）。Token 过期后需重新登录。

## 租户设置

### 查看租户信息

1. 登录后，进入 **租户设置** 页面
2. 查看当前租户的名称、标识和状态

[截图：租户设置页面]

### 修改租户信息

1. 在租户设置页面修改 **租户名称**
2. 点击 **保存**

## 团队成员管理

### 邀请成员

1. 进入 **租户设置** 页面
2. 在 **团队成员** 区域点击 **邀请成员**
3. 填写被邀请人信息：
   - **邮箱**：被邀请人的邮箱地址
   - **角色**：选择角色（见下方角色说明）
4. 点击 **确认邀请**

[截图：邀请成员界面]

### 角色权限说明

| 角色 | 权限 | 说明 |
|------|------|------|
| 管理员（admin） | 全部操作 | 管理租户设置、团队成员、知识库、聊天组件、API Key、LLM 配置 |
| 成员（member） | 知识库 + 任务 + 评测 | 上传管理文档、查看任务状态、配置评测集、调整聊天组件外观 |
| 只读（readonly） | 仅查看 | 查看文档列表和任务状态，不可修改 |

### 管理成员

- **修改角色**：在成员列表中，点击成员的角色下拉框，选择新角色
- **移除成员**：点击成员操作栏的 **移除** 按钮

> **注意**：不能移除自己，不能移除最后一个管理员。至少保留一名管理员。

## 安全机制

- **JWT 无状态鉴权**：管理后台使用 JWT Token 鉴权，Token 存储在浏览器 localStorage
- **多租户隔离**：所有数据按租户隔离，通过 TenantContext + AOP 自动过滤，不同租户之间数据不可见
- **密码安全**：密码使用哈希加密存储，不明文保存
- **每日调用限额**：V1 新增每日对话调用次数硬限制，防止滥用
