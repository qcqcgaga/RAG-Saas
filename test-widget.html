DocChat 聊天组件测试页

用于验证嵌入式聊天组件是否正常工作

📋 测试步骤

确保后端服务已启动（http://localhost:8080）

在 管理后台 注册/登录，确保知识库中有已处理的文档

进入 聊天组件 页面，复制嵌入脚本代码

将脚本粘贴到下方 <!-- 在此粘贴嵌入脚本 --> 位置，保存文件

在浏览器中打开此页面（必须通过 HTTP 服务器，不能直接用 file:// 协议打开）

右下角应出现聊天图标，点击图标输入问题验证对话功能

🔗 管理后台

访问 http://localhost:5173 管理文档和组件配置

⚠️ 注意：此页面必须通过 HTTP 服务器访问（如 python -m http.server）， 不能直接用 file:// 协议打开，否则浏览器会阻止加载跨域的 widget.js。
widget.js 加载失败！请确认后端服务已启动且 URL 正确。
